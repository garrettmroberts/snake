# Snake Game + Reinforcement Learning AI

## Getting started

- Ensure you have flask, flask_socketio, and torch installed.
- In terminal, run `flask --app app run --debug`
- Frontend will run at [http://127.0.0.1:5000](http://127.0.0.1:5000).
